# 股票数据分析练习项目
Python + Pandas + Matplotlib 股票数据处理与可视化练习

## 项目介绍
本项目使用真实模拟股票数据，练习 pandas 数据处理、时间序列分析、指标计算、条件筛选与数据可视化。

Python数据分析、量化交易入门练习。

## 实现功能
1. 数据读取与时间格式转换
   ，通过 Pandas 读取 CSV 数据，自动解析日期字段，对缺失值、异常值进行处理，确保数据完整性和准确性，为后续分析奠定基础
3. 股票分组排序
4. 核心指标计算
   ，基于 NumPy 和 Pandas 计算股票分析中常用的核心指标，可直接复用或修改参数（如均线窗口、计算周期）
5. 数据可视化
   通过 Matplotlib 绘制直观的图表，将分析结果可视化，支持保存为图片文件，便于展示和分析
6. 结果导出
   ，将清洗后的数据和计算的指标，导出为 CSV 文件，便于后续进一步分析或存档

## 技术栈
- Python 3.x
- Pandas
- Matplotlib
- NumPy
