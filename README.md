# 股票数据分析练习项目
Python + Pandas + Matplotlib 股票数据处理与可视化练习

## 项目介绍
本项目使用真实模拟股票数据，练习 pandas 数据处理、时间序列分析、指标计算、条件筛选与数据可视化。

Python数据分析、量化交易入门练习。

## 实现功能
1. 数据读取与时间格式转换
2. 多股票分组排序
3. 每日涨跌幅计算（pct_change）
4. 5日、20日均线计算
5. 放量上涨策略筛选（成交量>1.5倍均值 + 收盘>开盘）
6. 异常值检测（均值±2倍标准差）
7. 收盘价+均线走势图
8. 成交量柱状图
9. 收益率分布直方图

## 技术栈
- Python 3.x
- Pandas
- Matplotlib
- NumPy
